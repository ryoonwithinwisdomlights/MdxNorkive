<strong>🗺️ 전체 흐름 요약</strong>



```plain text
[작성 단계]  
 Notion에서 콘텐츠 작성  
       ↓
[변환 단계]  
 notion-to-md로 MDX 변환 (.mdx 파일 자동 생성)  
       ↓
[정적 분석 단계]  
 @content-collections/core로 타입 검증 + 슬러그 매핑  
       ↓
[렌더링 단계]  
 Next.js App Router에서 @content-collections/next로 렌더링
```


## <strong>✅ 1단계: 디렉토리 및 환경 준비</strong>


### <strong>📁 기본 디렉토리 구조</strong>


```plain text
/content
  └── /blog          ← 변환된 MDX 파일 저장 위치
      └── hello.mdx

/scripts
  └── notion-to-mdx.ts  ← 자동 변환 스크립트

/pages (or /app)
  └── /blog/[slug]/page.tsx

/content.config.ts     ← 콘텐츠 구성 정의
/next.config.ts        ← withContentCollections 적용
/blog.config.ts        ← 기존 Notion API 설정
```


```bash
npx content-collections install
```
